# WebSocketDemo
Rxjava2 + OkHttp webSocket 连接

使用IntentService方式连接WebSocket

连接成功之后，服务器告诉客户端成功，随后发送数据： 测试如下，自检服务每隔十秒钟自动发送一次Pong
给服务端，服务器相应的每隔十秒钟发送一次 #Ping# 给客户端保持连接！
